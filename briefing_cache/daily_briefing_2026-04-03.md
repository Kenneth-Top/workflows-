### 📊 核心观点摘要

今日数据显示，大模型行业呈现**“端侧模型爆发”**与**“头部智力争霸”**的双重格局。
1.  **小米 Mimo 现象级爆发**：`xiaomi/mimo-v2-pro` Token 消耗量突破 5.3 万亿（5331B），断层式领先，标志着端侧/生态侧模型落地取得重大突破。
2.  **Anthropic 暂登智力巅峰**：`Claude Opus 4.6` 以 1504 分刷新 LMArena 记录，并在消耗榜稳居前五，实现技术声誉与商业落地的双赢。

### 🏢 第一部分：核心厂商动态与竞争格局

#### 1. Anthropic：技术登顶，商业化稳健
*   **跑分动态**：根据 LMArena 最新数据，`claude-opus-4-6-thinking` 以 **1504分** 位居榜首，`claude-opus-4-6` 紧随其后（1499分）。这标志着 Anthropic 在推理与通用能力上已暂时超越 OpenAI 和 Google。
*   **市场表现**：消耗榜显示，`claude-opus-4.6` 与 `claude-sonnet-4.6` 的消耗量均突破 1.1 万亿 Token（分别为 1175B 和 1171B），分列第 5、6 位。高单价模型的高消耗验证了其 To B 端极高的付费意愿与粘性。

#### 2. OpenAI：跑分承压，流量缺席？
*   **跑分动态**：`gpt-5.4-high` 得分 1484，仅位列第 6，甚至低于 Google 的 Gemini 3.1 Pro 和 X-AI 的 Grok 4.20。这是 OpenAI 在高端模型统治力下降的明显信号。
*   **流量异动**：值得注意的是，在 Token 消耗 Top 15 榜单中，**未见 OpenAI 主力模型（如 GPT 系列）身影**。这可能意味着 OpenAI 的 API 流量被其他竞品分流，或者其主力流量集中在官方 App 而非聚合 API 市场，需警惕其市场份额被侵蚀的风险。

#### 3. Google：Gemini 3 系列全面渗透
*   **市场表现**：`gemini-3-flash-preview` 消耗量达 1122B（第 7 位），显示出极强的市场渗透力。
*   **技术实力**：`gemini-3.1-pro-preview` 跑分 1494（第 3 位），`gemini-3-pro` 跑分 1486（第 5 位）。Google 已形成“Flash 抢流量，Pro 争高端”的稳固阵型。

#### 4. 国产厂商：流量收割与价格战
*   **Minimax**：`minimax-m2.7` 消耗量 1534B（第 3 位），表现极其强劲，证明其模型在特定场景（如角色扮演、长文本）具有极高的用户粘性。
*   **DeepSeek**：`deepseek-v3.2` 消耗量 1393B（第 4 位），持续保持高热度，作为开源与低价策略的标杆，其流量基本盘稳固。
*   **阿里 Qwen**：`qwen3.6-plus-preview:free` 消耗量 1099B（第 8 位），依靠“免费”策略成功抢占大量长尾市场。
*   **智谱 GLM**：`glm-5-turbo` 消耗量 1035B（第 9 位），稳居第一梯队。

### 📈 第二部分：变化与动量分析

#### 🔥 用量激增：小米 Mimo 的“断层式”领先
*   **数据表现**：`xiaomi/mimo-v2-pro` 单周消耗 **5331.48B** Tokens，是第二名 Stepfun 的 3 倍以上。
*   **归因分析**：这种异常的激增通常源于**端侧设备的大规模OTA升级**或**特定生态应用（如小米汽车、智能家居）的全面接入**。这表明“模型+硬件/OS”的闭环模式正在产生远超纯 API 服务的调用规模。

#### 📉 价格与策略：免费策略依然有效
*   **Stepfun**：`stepfun/step-3.5-flash:free` 以 1575B 的消耗量位列第 2，证明在价格敏感型市场，免费策略依然是获取流量的最有效手段。
*   **阿里**：定价表中 `alibaba/tongyi-deepresearch-30b-a3b` 输入价格低至 **$0.09/M**（采样数据），结合 Qwen3.6 的高消耗，说明阿里正在通过极致性价比策略清洗市场。

#### 🏆 跑分黑马：X-AI Grok 4.20
*   `grok-4.20-beta1` 跑分达 **1491**，位列第 4，超越 GPT-5.4-high。结合其消耗量（`grok-4.1-fast` 排名第 10），X-AI 正在成为不可忽视的“第三极”力量。

### 🎨 第三部分：多模态专区

#### Vision 与多模态模型评测变动
*   **小米双模态霸榜**：消耗榜中 `xiaomi/mimo-v2-pro` (第1) 与 `xiaomi/mimo-v2-omni` (第12) 同时上榜。`omni` 后缀通常代表全模态（语音+视觉+文本）能力。这暗示**多模态交互正在成为端侧设备的标准配置**，且调用频率极高。
*   **Google Gemini 的多模态优势**：`gemini-3-flash-preview` 的高消耗与其原生的多模态能力密切相关。在 LMArena 榜单中，Gemini 系列在多模态任务（Vision）中通常具有统治力，支撑了其高流量。
*   **投资观察**：纯文本模型的流量增长已现疲态，未来的增长点在于 Vision 模型在端侧的落地。小米的数据证实了“多模态+硬件”的商业变现潜力远超纯软件 API。

### 📝 结论与投资建议

1.  **关注端侧落地**：小米 Mimo 的数据异动是今日最大亮点，建议重点关注具备“OS+硬件+模型”全栈能力的厂商，其流量天花板远高于纯软件厂商。
2.  **技术壁垒重塑**：Anthropic 暂时夺回技术王座，OpenAI 面临压力。投资应关注技术迭代动能强的标的，警惕技术护城河消失带来的估值回调。
3.  **流量变现逻辑**：Stepfun、Qwen 等通过免费/低价策略获取的海量流量（Top 10 占据 4 席），正在重塑 API 市场的定价体系。建议关注具备低成本推理架构（如 MoE）的厂商，他们将在价格战中存活并收割市场。
