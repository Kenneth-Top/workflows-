# 【每日大模型行业情报简报】

**日期：** 2026年3月16日
**编辑：** AI 行业投资顾问

---

### 📊 核心观点摘要

今日数据显示，大模型行业呈现**“用量向头部集中，技术向多模态演进”**的显著趋势。
1.  **MiniMax 异军突起**：`minimax-m2.5` 以近 2 万亿 Token 的周消耗量霸榜，成为当前 API 市场流量最大的模型。
2.  **Claude 登顶性能王座**：Anthropic 的 `claude-opus-4-6` 在 LMArena 跑分中突破 1500 分，超越 GPT-5.4 系列位列第一。
3.  **Google 强势反击**：Gemini 3 系列在流量与跑分双线告捷，`gemini-3-flash-preview` 流量激增，性价比优势明显。
4.  **OpenAI 市场份额警示**：在 API 消耗 Top 15 中，OpenAI 仅靠 `gpt-oss-120b` 和 `gpt-5-nano` 勉强入围，且排名靠后，面临严峻的存量竞争。

---

### 🏢 核心厂商深度分析

#### 1. Anthropic：技术护城河加深，高端模型需求稳健
*   **跑分动态**：`claude-opus-4-6` 与 `claude-opus-4-6-thinking` 以 **1503 分** 并列 LMArena 榜首，显著领先于 Grok 4.20 (1496分) 和 GPT-5.4-high (1485分)。这表明 Anthropic 在复杂推理任务上已建立短期技术代差。
*   **流量表现**：`claude-sonnet-4.6` (974.3B) 和 `claude-opus-4.6` (908.4B) 分列消耗榜第 5、6 位。Sonnet 版本作为主力生产模型，用量略高于 Opus，显示企业用户在“性能与成本”平衡上的理性选择。

#### 2. MiniMax & 阶跃星辰：国产模型流量爆发
*   **MiniMax**：`minimax-m2.5` 以 **1996.33B** (近 2 万亿) Token 消耗量位居榜首，远超第二名。推测其可能在大规模 B 端应用或特定高并发场景（如语音合成、Agent 任务）中实现了突破性落地。
*   **阶跃星辰**：`step-3.5-flash:free` 以 **1483.75B** 的消耗量紧随其后。模型名称中的 `free` 标签暗示其通过免费策略成功抢占了大量市场份额，属于典型的“以价换量”策略。

#### 3. Google：Gemini 3 系列全面发力
*   **流量激增**：`gemini-3-flash-preview` 消耗量达 **1128.77B**，排名第四。结合其 LMArena **1474 分**的成绩，说明该模型在保持高性能的同时，通过 Flash 架构实现了极高的吞吐量，正在快速蚕食 GPT 的中端市场份额。
*   **价格竞争力**：虽然采样数据中未直接列出 Gemini 3 价格，但对比同系列 `gemini-2.5-flash-lite` 的定价策略，Google 正在通过极具竞争力的价格体系维持高增长。

#### 4. DeepSeek & Moonshot：存量竞争中的稳健派
*   **DeepSeek**：`deepseek-v3.2` 消耗量 **1169.89B**，稳居第三。在价格战激烈的环境下，DeepSeek 依然保持了极高的调用粘性，证明其技术生态已形成护城河。
*   **Moonshot (Kimi)**：`kimi-k2.5` 消耗量 **628.99B**，排名第八。作为主要面向 C 端的长文本模型，其 API 流量能进入前十，显示出其在 Agent 和长上下文应用场景中的不可替代性。

#### 5. OpenAI：面临“空心化”挑战
*   **数据警示**：在消耗量前 15 中，OpenAI 仅 `gpt-oss-120b` (491.4B, 排名13) 和 `gpt-5-nano` (354.2B, 排名15) 入围。
*   **跑分落后**：`gpt-5.4-high` 跑分 1485，已被 Claude Opus 4.6 和 Grok 4.20 超越。OpenAI 在高端模型（GPT-5 系列）的 API 调用上似乎正在流失给 Anthropic 和 Google，而中低端市场则受到国产模型的冲击。

---

### 🖼️ 多模态与视觉专区

**市场格局变动：**
虽然本次 LMArena 数据主要展示文本得分，但结合流量数据可洞察多模态趋势：
*   **Gemini 3 Flash 的多模态渗透**：`gemini-3-flash-preview` 的高流量（1128B）与其原生多模态能力高度相关。作为榜单中跑分最高（1474）的 Flash 级模型，它正在成为处理图文混合任务的首选性价比模型。
*   **Claude Vision 能力隐含分析**：`claude-sonnet-4.6` 的高消耗量（974B）部分归功于其强大的视觉理解能力。在 Opus 版本登顶文本榜的背景下，Sonnet 版本承担了大量包含图表分析、文档识别的工业级多模态任务。
*   **开源多模态动态**：采样数据中出现了 `allenai/molmo-2-8b`，定价仅为 Input $0.19 / Output $0.18，虽然流量未进 Top 15，但极低的价格预示着开源多模态模型正在尝试降低门槛。

---

### 💰 价格与成本动态

*   **价格战白热化**：阿里云的 `alibaba/tongyi-deepresearch-30b-a3b` 提供了惊人的低价：**Input $0.088 / Output $0.449**，且缓存命中率高达 **97.7%**。这表明国产模型在推理成本优化上已做到极致，通过高缓存命中率大幅降低实际使用成本。
*   **免费策略常态化**：`step-3.5-flash:free` 的高流量证明，对于中等参数量的模型，“免费”已成为有效的市场进入手段。
*   **高端模型溢价维持**：尽管中低端市场价格崩塌，但 Anthropic 的 Opus 系列和 OpenAI 的 GPT-5 High 系列依然维持高定价逻辑，说明顶尖智力依然稀缺且昂贵。

---

### 📈 投资建议与风险提示

1.  **关注 Anthropic 生态**：Claude Opus 4.6 的技术领先优势明显，建议关注基于 Claude 构建的垂直应用层投资机会。
2.  **警惕 OpenAI 增长放缓**：OpenAI 在 API 流量榜单中的缺席是一个危险信号，需重新评估其护城河深度。
3.  **国产模型出海机会**：MiniMax 和 StepFun 的流量数据证明，中国大模型厂商在工程化落地和成本控制上已具备全球竞争力，建议关注其出海进展。
4.  **性价比标的**：Google Gemini 3 Flash 和 DeepSeek V3.2 是当前“性能/成本”比率最优的选择，适合作为底层基座模型的投资标的。
