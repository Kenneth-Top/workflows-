**【每日大模型行业情报简报】**

**日期：** 2026年04月23日
**分析师：** AI 行业投资顾问

---

### **一、 核心摘要**

今日数据显示，大模型行业竞争格局发生显著位移。**Anthropic** 凭借 Claude 系列在 API 调用量与跑分榜单上实现“双冠王”，尤其是 `Claude-Sonnet-4.6` 的用量激增，显示出极强的市场渗透力。**国产力量**表现抢眼，**DeepSeek** 与 **Minimax** 在 Token 消耗量上稳居第一梯队，**小米** 凭借 `Mimo-V2-Pro` 意外杀入前四，成为本月最大黑马。相比之下，**OpenAI** 的 GPT-5.4 在调用量与跑分上均面临来自 Anthropic 和 Google 的强力挤压。

---

### **二、 核心厂商动态与市场格局**

#### **1. Anthropic：全面领跑，技术与商业化双丰收**
*   **用量激增**：`Claude-Sonnet-4.6` 以 **1623.7 Billion** 的 Token 消耗量位居榜首，大幅领先第二名。`Claude-Opus-4.6` 与 `4.7` 版本合计贡献超 1800B 的调用量，表明其新旧版本迭代平滑，企业端粘性极高。
*   **跑分霸榜**：在 LMArena 榜单中，`Claude-Opus-4-7-Thinking` 以 **1503分** 夺冠，`4-6-Thinking` 紧随其后。Anthropic 已在推理能力上构建起短期不可逾越的护城河。
*   **投资观点**：Anthropic 目前在“高性能推理”与“高频次调用”两个维度均处于统治地位，其 Sonnet 模型已成为行业事实上的生产力标准。

#### **2. 国产模型：DeepSeek 稳守性价比，Minimax 双模型发力**
*   **DeepSeek**：`DeepSeek-V3.2` 消耗量达 **1473.4 Billion**，稳居第二。这印证了其“极致性价比”策略在开发者市场持续生效，并未因头部竞争而流失份额。
*   **Minimax**：成为榜单中唯一拥有两款模型进入前 7 的厂商。`Minimax-M2.5` (1136.2B) 与 `Minimax-M2.7` (1003.3B) 合计流量惊人，显示出其模型矩阵策略成功覆盖了不同层级的用户需求。

#### **3. Google：Gemini 3 系列攻势凌厉**
*   **数据表现**：`Gemini-3-Flash-Preview` 消耗量达 **1313.4 Billion**，排名第三。其跑分方面，`Gemini-3.1-Pro-Preview` 以 **1493分** 位列总榜第五，仅次于 Claude 系列和 Muse-Spark。
*   **分析**：Google 正通过激进的迭代（Gemini 3 系列）抢占 OpenAI 的市场份额，Flash 版本的高流量显示出其在轻量级任务中的优势。

#### **4. OpenAI：面临多方围剿**
*   **数据警示**：`GPT-5.4` 的 Token 消耗量仅为 **704.4 Billion**，排名跌至第 11 位，远低于 Claude Sonnet 和 DeepSeek。跑分方面，`GPT-5.4-High` 为 **1482分**，已被 Anthropic 拉开 20 分差距，且被 Google 追平。
*   **投资观点**：OpenAI 的市场统治力正在被稀释，急需在下一代模型或应用层寻找新的增长点。

---

### **三、 黑马观察：小米 Mimo 强势入局**

*   **现象级数据**：`Xiaomi/Mimo-V2-Pro` 以 **1226.3 Billion** 的消耗量跃居榜单第四，超越了 GPT-5.4 和 Grok 等知名模型。
*   **归因分析**：这极可能得益于小米庞大的终端生态整合。作为端侧/云端协同的典型代表，Mimo 的高流量预示着“手机厂商+大模型”的商业模式正在跑通，终端入口优势开始转化为 API 调用优势。

---

### **四、 价格战与成本分析**

*   **阿里通义**：`Tongyi-DeepResearch-30b-A3B` 输入价格低至 **$0.089/M**，输出 **$0.45/M**，价格极具破坏力，显示出阿里意图通过低价策略抢占长尾开发者市场。
*   **Amazon Bedrock**：`Nova-Lite-V1` 与 `Micro-V1` 维持极低价格（输入 $0.034-$0.059/M），云厂商通过自研模型降低生态内使用成本的意图明显。
*   **趋势**：头部模型价格趋于稳定，但中腰部及云厂商自研模型价格战仍在继续，API 服务正在向“自来水”化演进。

---

### **五、 多模态专区**

*   **Vision 模型异动**：
    虽然本次 LMArena 榜单未单独列出多模态子榜，但从 Token 消耗数据来看，**多模态已成为标配**。
    *   **小米 Mimo-V2-Pro**：名称即暗示其多模态属性，其高达 1226B 的调用量证明了多模态在移动端场景（图文理解、拍照搜索）的巨大需求。
    *   **Google Gemini 3 Flash**：作为多模态原生模型，其高消耗量也部分归功于多模态处理能力。
*   **投资建议**：单纯的文本模型已难以满足当前应用需求。投资者应重点关注在 Vision 能力上有突破且成本控制优秀的模型（如 Gemini 3 Flash, Mimo V2），这将是下一阶段应用爆发的核心驱动力。

---

### **六、 风险提示**

1.  **数据来源局限**：本报告基于特定聚合平台的 API 数据，可能未能完全覆盖 OpenAI 官方渠道及私有化部署的调用量，GPT-5.4 的实际市场占有率可能被低估。
2.  **模型迭代风险**：榜单变动剧烈（如 Claude 4.6/4.7 快速更替），技术领先优势维持时间缩短，投资需警惕技术路线迭代过快带来的资产贬值风险。
