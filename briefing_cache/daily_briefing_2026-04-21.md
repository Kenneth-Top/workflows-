# 【每日大模型行业情报简报】

**日期：** 2026年04月21日
**编辑：** AI 行业投资顾问

---

### 01. 核心摘要

今日数据显示，大模型行业格局发生显著位移。**Anthropic 凭借 Claude 4.6 系列在用量上实现突破，首次在 Token 消耗总量上压制 OpenAI**；国产阵营中，**DeepSeek V3.2 与小米 Mimo-V2-Pro** 表现出极强的市场穿透力，尤其是小米模型的异军突起值得高度关注。跑分方面，Claude Opus 4.7 Thinking 版本稳居榜首，OpenAI GPT-5.4 在竞技场得分上落后 Anthropic 约 22 分，技术代差隐现。

---

### 02. 核心厂商动态与用量分析

#### **Anthropic：全面领跑，Sonnet 成为“新基建”**
*   **数据表现**：`claude-sonnet-4.6` 以 **1593.28B** 的 Token 消耗量位居全球第一，甚至超过了 DeepSeek V3.2。旗舰模型 `claude-opus-4.6` 消耗量也高达 **1353.00B**，排名第三。
*   **分析师点评**：Anthropic 已成功构建“双轮驱动”模式。Sonnet 4.6 凭借极高的性价比（推测）承接了大量日常开发与生产任务，而 Opus 4.6 则锁定了高端复杂推理市场。两者合计消耗量远超 OpenAI 同类产品，显示出开发者生态的重心正在向 Claude 系列迁移。

#### **DeepSeek & 国产力量：DeepSeek 稳居第二，小米成最大黑马**
*   **数据表现**：`deepseek-v3.2` 消耗量 **1478.30B**，稳居榜单第二，与 Claude Sonnet 差距微小。
*   **黑马预警**：`xiaomi/mimo-v2-pro` 以 **1315.46B** 的消耗量位列第四，这一数据极其惊人。作为对比，Google Gemini 3 Flash Preview 仅为 1313.70B。小米在端侧与云侧模型的融合策略似乎取得了巨大成效，其用量激增可能源于小米硬件生态（手机/IoT）的深度集成。
*   **Minimax**：`minimax-m2.5` 与 `minimax-m2.7` 分别位列第 6、7 名，合计消耗量超过 2200B，显示出其在特定场景（如角色扮演、长文本）的稳固基本盘。

#### **OpenAI：份额承压，GPT-5.4 未达预期峰值**
*   **数据表现**：`openai/gpt-5.4` 消耗量 **649.93B**，仅排名第 11 位。
*   **分析师点评**：这是极其罕见的排名低位。GPT-5.4 的消耗量不足 Claude Sonnet 4.6 的一半。结合跑分数据，OpenAI 目前处于“技术爬坡期”或“产品迭代空窗期”的劣势。虽然 `openai/gpt-oss-120b`（推测为开源版或轻量版）位列第 15，但未能撑起整体大盘。

#### **Google：Gemini 3 Flash 预览版表现强劲**
*   **数据表现**：`google/gemini-3-flash-preview` 消耗量 **1313.71B**，排名第 5，与小米 Mimo 和 DeepSeek 处于同一梯队。
*   **分析师点评**：Google 的“Flash”策略（高速、低成本）在 API 市场依然有效，成功守住了第一梯队的席位。

### 03. 多模态专区

*   **Vision 模型竞争格局**：
    虽然数据库未单独列出 Vision 跑分，但从 Token 消耗来看，支持多模态输入的模型占据了榜单前列。
    *   **Gemini 3 Flash Preview**：作为原生的多模态模型，其高消耗量（1313.71B）印证了多模态处理（特别是视频/图像理解）正在成为 API 调用的主流场景。
    *   **Xiaomi Mimo-V2-Pro**：名称中的 "V" 通常代表 Vision，其激增的用量暗示端侧多模态交互（如手机摄影视觉分析、OCR等）正在爆发。
    *   **Claude 系列**：Claude 4.6 系列强大的图表与图像推理能力是其区别于 GPT-5.4 的关键优势，这也是其消耗量激增的核心驱动力之一。

### 04. 跑分与定价深度解读

#### **LMArena 竞技场：Anthropic 暂时封神**
*   **榜单统治**：`claude-opus-4-7-thinking` 以 **1504** 分高居榜首，`claude-opus-4-6-thinking` 紧随其后（1502分）。
*   **OpenAI 追赶态势**：`gpt-5.4-high` 得分 **1482**，与 Claude Opus 4.7 存在约 **22 分**的差距。在竞技场评价体系中，这属于显著代差。
*   **Google 布局**：`gemini-3.1-pro-preview` 得分 1493，成功卡位在 Claude 与 GPT 之间，显示出 Gemini 3.0/3.1 系列在推理能力上的长足进步。

#### **API 定价策略：阿里与 Amazon 发起价格战**
*   **阿里云**：`alibaba/tongyi-deepresearch-30b-a3b` 定价极具侵略性，输入 **$0.09/M**，输出 **$0.45/M**。这一价格远低于市场平均水平，意在通过低价抢占长尾开发者和科研市场。
*   **Amazon Bedrock**：`amazon/nova-lite-v1` 输入低至 **$0.06/M**，`nova-micro-v1` 更是低至 **$0.03/M**。Amazon 正在利用其基础设施优势，将基础模型价格压至“白菜价”，以此巩固 AWS 的生态粘性。
*   **高价区**：AI21 的 `jamba-large-1.7` 输入 $2.0/M，输出 $8.0/M，在当前降价潮中显得格格不入，除非其有特定垂直领域的不可替代性，否则市场份额堪忧。

### 05. 投资建议与风险提示

1.  **关注端侧模型爆发**：小米 `Mimo-V2-Pro` 的数据异动提示我们，手机厂商主导的端云结合模型正在成为不可忽视的力量，建议关注相关硬件生态链的投资机会。
2.  **OpenAI 份额预警**：GPT-5.4 在用量和跑分上的双重落后，提示 OpenAI 可能面临自 GPT-4 发布以来最大的竞争压力。需密切关注其后续版本（如 GPT-5.5 或 Turbo 版）的发布节奏。
3.  **价格战红利**：阿里和 Amazon 的超低定价策略将加速中小模型厂商的出清，利好应用层开发者，但利空纯模型层初创公司。
