# 【每日大模型行业情报简报】

**日期：** 2026年3月13日
**编辑：** AI 行业投资顾问

---

### 核心观点摘要

今日数据显示，大模型行业呈现**“双雄争霸，国产突围”**的格局。
1.  **用量侧**：MiniMax 凭借 `minimax-m2.5` 实现爆发式增长，Token 消耗量突破 2.2 万亿，断层领先；阶跃星辰 凭借免费策略跃居第二。
2.  **性能侧**：Anthropic `Claude Opus 4.6` 重夺 LMArena 榜首，以 1503 分压制 Grok 与 Gemini，OpenAI GPT-5.4 暂居第四。
3.  **价格侧**：阿里通义与 Amazon 在低成本赛道极致内卷，输入价格击穿 $0.1/M 关口。

> **注**：今日外部网络搜索资讯主要涉及游戏平台 Roblox 相关内容，与 AI 大模型核心赛道无直接关联。本简报以下分析完全基于数据库客观运行数据与定价采样。

---

### 一、 核心厂商动态与用量分析

#### 1. MiniMax (Hailuo)：流量之王，数据断层第一
*   **数据表现**：`minimax/minimax-m2.5` 近7日 Token 消耗高达 **2253.16 B (十亿)**，是第二名 StepFun 的近两倍。
*   **分析师点评**：MiniMax 已成为当前用户调用量最大的模型之一。其 M2.5 模型在多模态与长文本上的优势可能正在转化为巨大的 C 端与 B 端调用粘性。这种量级的消耗显示出其产品（海螺 AI 等）已进入高频使用区间。

#### 2. 阶跃星辰：免费策略奏效，跻身第二梯队
*   **数据表现**：`stepfun/step-3.5-flash:free` 消耗量达 **1272.04 B**。
*   **分析师点评**：作为标注为 `free` 的模型，其消耗量高居第二，证明“免费/试用”策略在抢占市场份额方面依然极其有效。StepFun 正在通过激进的推广策略快速积累用户数据。

#### 3. Google Gemini：Flash 系列成主力
*   **数据表现**：`google/gemini-3-flash-preview` (1166.79 B) 与 `google/gemini-2.5-flash` (629.93 B) 合计消耗量巨大。
*   **分析师点评**：Google 的模型迭代速度极快，Gemini 3 Flash Preview 的消耗量已超越 DeepSeek V3.2。Google 正在通过高性能的 Flash 版本（兼顾速度与成本）在 API 市场抢占开发者生态。

#### 4. Anthropic：高端模型需求稳健
*   **数据表现**：`claude-sonnet-4.6` (933.43 B) 与 `claude-opus-4.6` (902.10 B) 紧随其后。
*   **分析师点评**：不同于 Google 和 MiniMax 的“流量打法”，Anthropic 在高单价、高性能模型上依然保持了极高的调用量，说明其在复杂任务处理（如 Coding、长文本分析）上拥有极高的不可替代性。

#### 5. DeepSeek：V3.2 保持高热度
*   **数据表现**：`deepseek/deepseek-v3.2` 消耗量为 **1084.91 B**。
*   **分析师点评**：DeepSeek 依然维持在千亿级 Token 消耗梯队，证明了其开源生态与低价策略的长尾效应。

### 二、 跑分与性能风向标

根据最新 LMArena 综合跑分排名，行业格局发生微妙变化：

| 排名 | 模型 | 分数 | 分析师解读 |
| :--- | :--- | :--- | :--- |
| **1** | **Claude Opus 4.6** | **1503** | **重夺王座**。Anthropic 的最新旗舰模型在文本理解与推理上确立了当前 SOTA (State of the Art)。 |
| **2** | **Grok 4.20 Beta1** | **1496** | xAI 进步神速，Grok 4.20 已逼近第一梯队，仅落后 Claude 7分。 |
| **3** | **Gemini 3.1 Pro** | **1492** | Google 旗舰模型稳居前三，与 Grok 差距极小。 |
| **4** | **GPT-5.4 High** | **1485** | OpenAI 暂居第四。虽然分数依然极高，但在“最强模型”的争夺上，OpenAI 目前面临 Anthropic 与 Google 的双重压力。 |
| **5** | **Gemini 3 Flash** | **1474** | **亮点**：Flash 模型跑分进入 Top 5，意味着用户可以用极低的成本获得接近旗舰模型的体验，这是 Google 最大的杀手锏。 |

**投资启示**：OpenAI 的技术领先优势正在被 Anthropic 和 Google 追平甚至反超。市场不再是 GPT 的一家独大，多极化格局利好算力基础设施提供商（推理需求多样化）及应用层开发者（议价权提升）。

### 三、 API 定价与成本分析

价格战在中小参数模型与特定场景模型中愈演愈烈：

*   **极致性价比区间**：
    *   **Amazon Nova Micro**：输入价格低至 **$0.035/M**，输出 **$0.137/M**。AWS 正在利用其云生态优势压榨推理成本。
    *   **阿里通义**：`alibaba/tongyi-deepresearch-30b-a3b` 输入价格 **$0.089/M**，且 Cache Hit Rate 高达 **94.95%**。这表明阿里在长上下文缓存优化上技术成熟，极适合高重复性查询场景（如 RAG）。

*   **高价区间**：
    *   **AI21 Jamba Large**：输入 $2.0/M，输出 $8.0/M。在开源模型价格极低的背景下，此类高价模型必须提供极具差异化的企业级功能（如极长上下文、零幻觉）才能生存。

### 四、 多模态专区

虽然数据库未直接提供多模态跑分细节，但结合模型命名与消耗数据，我们观察到以下趋势：

1.  **Gemini 3 Flash 的多模态渗透**：
    `gemini-3-flash-preview` 拥有超过 1100B 的消耗量。鉴于 Gemini 系列原生的多模态能力，推测大量调用包含图像理解任务。其 1474 的高跑分证明**多模态模型已不再牺牲文本性能**。

2.  **MiniMax 的潜在多模态爆发**：
    `minimax-m2.5` 的惊人流量可能与其强大的视频/图像生成与理解能力有关。MiniMax 一向在多模态（尤其是视频生成）上有技术积累，高 Token 消耗可能不仅源于文本，更源于多模态交互的“高带宽”特性。

3.  **投资建议**：
    建议关注 **Vision-Text 融合模型**的商业化落地。随着 Gemini 3 Flash 等模型将多模态成本拉低至文本水平，传统的 OCR、基础图像分析 SaaS 将面临被模型能力替代的风险。

---

### 五、 总结与预测

*   **流量格局**：MiniMax 和 StepFun 的崛起显示了中国 AI 模型在用户侧的强劲增长，OpenAI 在 API 流量榜上的排名（GPT-OSS-120b 仅排第 11）与其声量存在一定错位，显示出市场正在分化为“高性能闭源”与“高性价比/开源”两大阵营。
*   **技术风向**：Claude Opus 4.6 登顶标志着“推理能力”成为新的竞争高地。Flash 模型跑分进入前五，预示着 2026 年将是“高性能模型普惠化”的一年。
*   **风险提示**：今日搜索资讯缺失，外部市场情绪不明。建议投资者重点关注下周各大厂商的模型更新节奏，特别是 OpenAI 针对 GPT-5.4 的后续优化版本。
