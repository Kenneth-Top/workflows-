API 定价数据显示，行业已进入“极致性价比”阶段，尤其是国产模型正在重塑价格锚点。

*   **阿里通义**：`tongyi-deepresearch-30b-a3b` 的输入价格低至 **$0.089/M**，输出 **$0.449/M**。这一价格对于 30B 参数量的模型极具破坏力，旨在通过低价抢占开发者生态入口。

## 4. 多模态专区

本期多模态领域出现显著的结构性变化，**视觉与全能模型正在吞噬传统文本模型的市场份额**。

*   **数据异动**：Token 消耗榜第一名 **Xiaomi Mimo-V2-Pro** (4671.7 B) 和第 13 名 **Xiaomi Mimo-V2-Omni** (635.0 B) 均为多模态/全能模型。Mimo-V2-Pro 的消耗量远超传统纯文本模型（如 GPT 系列、Claude 系列），这暗示了**端侧多模态交互（如视觉搜索、实时视频分析）可能正在迎来“iPhone 时刻”**。
*   **技术归因**：
    *   **Vision 能力下沉**：多模态不再是高端模型的专属，中端甚至端侧模型（如 Mimo 系列）已具备强大的视觉理解能力。
    *   **应用场景拓展**：高消耗量表明用户不再仅满足于文本生成，而是大量使用图像理解功能。这将对仅支持文本的旧模型构成降维打击。
*   **投资建议**：重点关注在端侧多模态落地方面有硬件优势的厂商（如小米、华为），以及提供高效多模态推理算力支持的底层设施。
