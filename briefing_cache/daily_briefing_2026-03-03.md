# 【每日大模型行业情报简报】

**日期：** 2026年3月3日
**编辑：** AI 行业投资顾问

---

## 1. 核心摘要

今日大模型市场呈现**"量价背离，格局重塑"**的显著特征。核心监测数据如下：

*   **用量霸主易位：** MiniMax 的 `minimax-m2.5` 以 1857B Token 消耗量断层领先，成为当前全网最繁忙模型，远超 Google 和 DeepSeek。
*   **跑分王座更迭：** Anthropic 凭借 `claude-opus-4-6-thinking` 以 1503 分登顶 LMArena，在代码与综合能力上全面超越 OpenAI 的 GPT-5.2 系列。
*   **性价比战况：** 阿里巴巴 `tongyi-deepresearch` 系列展现出极高的缓存命中率（>90%）与极低的单价，正在中腰部市场快速渗透。

> **注：** 今日外部搜索资讯主要涉及税务法规（Simples Nacional），与 AI 行业无直接关联，本简报完全基于核心数据库客观数据进行分析。

---

## 2. 核心厂商深度分析

### 2.1 Anthropic：技术登顶，新旧交替
*   **跑分表现：** `claude-opus-4-6-thinking` 与 `claude-opus-4-6` 并列第一（Score: 1503），在代码专项排名中霸榜前二。这标志着"Thinking"（思维链）模式已成为冲击高分的关键架构。
*   **流量动向：** 尽管跑分登顶，但在 API 消耗量上，`claude-opus-4.6`（743B）与 `claude-sonnet-4.6`（734B）虽位居前列，但总量仍低于 MiniMax。值得注意的是，旧版 `claude-sonnet-4.5` 仍有 612B 的存量消耗，显示用户正在向 4.6 版本迁移中，但尚未完全完成切换。

### 2.2 MiniMax：流量黑洞，规模制胜
*   **数据亮点：** `minimax/minimax-m2.5` 单周消耗 **1857.4B Tokens**，几乎是第二名 Google Gemini 的 1.5 倍。
*   **投资解读：** 在跑分榜未进前 20 的情况下，MiniMax 的消耗量激增，极有可能在**C端超级应用**或**特定垂直行业（如游戏、社交）**中获得了爆发式集成。这种"跑分不高但用量极大"的现象，暗示其模型在特定场景下的性价比或延迟表现具有压倒性优势。

### 2.3 Google：激进迭代，Flash 系列成主力
*   **模型动态：** `gemini-3-flash-preview` 消耗量达 1191B，位居第二。新版 `gemini-3.1-pro-preview` 跑分紧追 Claude 之后（1500分），显示出强大的技术追赶能力。
*   **策略分析：** Google 正在通过 "Flash" 系列以低价换取市场份额，旧版 `gemini-2.5-flash` 系列消耗量已显著回落，用户向 v3 版本迁移速度极快。

### 2.4 DeepSeek & Moonshot：国产双雄稳居第一梯队
*   **DeepSeek：** `deepseek-v3.2` 消耗量 906B，稳居第三。在保持高消耗的同时，其价格策略依然激进，维持了极高的开发者粘性。
*   **Moonshot (Kimi)：** `kimi-k2.5` 消耗量 832B，且 `kimi-k2.5-thinking` 进入了 LMArena 前 20（Rank 19）。这表明 Kimi 不仅在应用层表现优异，在基座模型能力上也已跻身全球主流行列，特别是在 Coding 能力上（Rank 12）表现优异。

### 2.5 OpenAI：相对沉寂，等待反击
*   **现状：** 在消耗榜前 15 中，仅 `gpt-oss-120b` 位列末位（376B），且该模型疑似为开源/实验性模型。主力商业模型（如 GPT-5 系列）未出现在高频消耗榜单中。
*   **跑分：** `gpt-5.2-chat-latest` 以 1481 分位列第六，已被 Claude 和 Gemini 的新一代模型超越。OpenAI 目前处于产品发布周期的相对低谷期，市场份额正被 Anthropic 和 Google 快速蚕食。

## 3. 价格与成本效率监测

今日价格监测显示，开源生态与闭源生态的定价策略两极分化严重。

*   **阿里云激进定价：**
    `alibaba/tongyi-deepresearch-30b-a3b` 的输入价格仅为 **$0.09/M**，输出 **$0.45/M**。更关键的是，其 **Cache Hit Rate（缓存命中率）高达 90.7%**。这意味着在实际生产环境中，该模型的边际成本极低，极具价格竞争力，适合大规模低成本部署。

*   **免费模型流量效应：**
    榜单中 `arcee-ai/trinity-large-preview:free` 和 `stepfun/step-3.5-flash:free` 分别录得 622B 和 511B 的高消耗量。这验证了"免费/开源"策略依然是获取大量长尾开发者的最有效手段。

*   **Amazon Bedrock 入局：**
    `amazon/nova-lite-v1` 提供了极具竞争力的价格（Input $0.06/M, Output $0.24/M），AWS 正在通过 Nova 系列模型试图降低企业使用大模型的门槛。

## 4. 多模态专区

本周期内多模态模型呈现"隐性爆发"态势，虽然纯文本模型仍占据跑分榜首，但多模态能力已深度整合进主力模型。

*   **Vision 模型定价与性能：**
    `allenai/molmo-2-8b` 作为知名的多模态模型，其 API 定价已下探至 Input $0.19/M，显示出多模态推理成本正在快速下降，接近文本模型水平。

*   **Gemini 的多模态渗透：**
    消耗量极高的 `gemini-3-flash-preview` 原生支持多模态。其高达 1191B 的 Token 消耗量中，推测包含了相当比例的图像/视频理解任务。这表明**多模态已不再是"附加功能"，而是高流量模型的"标配"**。

*   **投资建议：** 关注多模态在工业质检、内容审核等场景的落地。随着 Molmo 等模型价格下降，视觉大模型的应用成本壁垒已被打破。

## 5. 投资建议与风险提示

**投资建议：**
1.  **关注 Anthropic 产业链：** Claude Opus 4.6 在代码和逻辑推理上的登顶，将吸引更多企业级开发者迁移，建议关注相关 API 代理、企业级 Agent 解决方案提供商。
2.  **MiniMax 现象级增长：** MiniMax 的高流量数据背后可能隐藏着新的应用形态（如 AI 陪伴、UGC 工具），建议重点关注其商业化落地路径。

**风险提示：**
1.  **OpenAI 市占率下滑风险：** GPT 系列在跑分和消耗量上的双重落后是一个危险信号，需警惕相关生态合作伙伴的业务波动风险。
2.  **价格战持续：** 阿里巴巴等厂商的高缓存命中率与低定价策略，将进一步压缩中小模型厂商的利润空间，行业出清加速。
