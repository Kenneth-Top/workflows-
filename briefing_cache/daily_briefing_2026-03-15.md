### 01. 核心摘要

今日数据显示，大模型行业呈现**“用量头部集中，跑分座次重排”**的显著特征。

1.  **国产模型流量霸榜**：MiniMax 与 Stepfun 在 Token 消耗量上实现反超，力压海外巨头，显示出极强的 C 端与 API 渗透力。
2.  **Anthropic 登顶性能王座**：Claude Opus 4.6 以 1503 分刷新 LMArena 记录，OpenAI GPT-5 系列在跑分上暂时处于追赶态势。
3.  **免费/低价策略奏效**：Step-3.5-flash 等免费模型占据流量榜第二，验证了“免费增值”模式在当前阶段的获客有效性。

---

### 02. 核心厂商动态与流量分析

#### **MiniMax：流量之王，M2.5 模型统治力显现**
*   **数据表现**：`minimax/minimax-m2.5` 以 **206.7 Billion** 的 Token 消耗量位居榜首，且大幅领先第二名（约 650B 的差距）。
*   **分析师点评**：MiniMax 已成为当前全球用量最大的模型之一。其 M2.5 模型的高流量不仅源于强大的生成能力，更得益于其在 C 端应用（如海螺 AI）及开发者生态中的激进推广。这种断层式的领先表明 MiniMax 正在承接大量原本属于 GPT 系列的推理需求。

#### **阶跃星辰：免费策略的巨大胜利**
*   **数据表现**：`stepfun/step-3.5-flash:free` 消耗量达 **141.8 Billion**，排名第二。
*   **分析师点评**：作为榜单前五中唯一的“免费”标签模型，Stepfun 的数据极具战略意义。这表明在价格战激烈的当下，零成本接入对开发者具有致命吸引力，成功抢占了长尾应用与高频低敏场景的市场份额。

#### **DeepSeek：国产开源中坚，稳居第一梯队**
*   **数据表现**：`deepseek/deepseek-v3.2` 消耗量 **114.3 Billion**，排名第三。
*   **分析师点评**：DeepSeek V3.2 保持了极高的调用热度，证明了其“高性能+低价格”策略的长期有效性。在开源生态中，DeepSeek 已成为事实上的行业标准替代品。

#### **Google：Gemini 3.0 预览版强势增长**
*   **数据表现**：`google/gemini-3-flash-preview` 消耗量 **113.9 Billion**，紧随 DeepSeek 之后。
*   **分析师点评**：Google Gemini 3.0 Preview 版本展现出极强的市场渗透力。相比旧版 Gemini 2.5 Flash（61.8B），新版本流量近乎翻倍，说明 Google 在模型迭代速度和多模态能力上的投入正在转化为实际的市场份额。

#### **Anthropic：双模驱动，高端市场稳固**
*   **数据表现**：`claude-sonnet-4.6`（95.9B）与 `claude-opus-4.6`（88.1B）分别位列第 5、6 名。
*   **分析师点评**：Anthropic 是榜单前十里唯一一家同时拥有两个主力模型且流量均衡的厂商。Sonnet 负责走量，Opus 负责高端任务，产品矩阵健康度极高。

#### **OpenAI：面临围剿，流量分散**
*   **数据表现**：榜单中未见 GPT-4o 或 GPT-5 主力版本进入前五。`openai/gpt-oss-120b`（52.2B）和 `openai/gpt-5-nano`（35.1B）排名靠后。
*   **分析师点评**：OpenAI 的流量结构呈现碎片化，主力模型可能因定价较高或被竞品分流，导致在 API 调用量上暂时落后于国产厂商。GPT-5-nano 的出现显示 OpenAI 正在尝试边缘端/低成本模型策略，但目前尚未形成统治级流量。

### 03. 性能跑分与竞争格局

根据 LMArena 最新数据，**模型能力天花板正在快速抬升**：

*   **Anthropic 暂居第一**：`claude-opus-4-6` 及其 Thinking 版本双双取得 **1503** 的高分，成为当前地表最强模型。
*   **Grok 与 Google 紧随其后**：X-AI 的 `grok-4.20-beta1`（1496分）和 Google 的 `gemini-3.1-pro-preview`（1492分）表现优异，差距仅在毫厘之间。
*   **OpenAI 稍显落后**：`gpt-5.4-high` 得分 1485，排名中游。虽然差距不大，但在“跑分军备竞赛”中，OpenAI 暂时失去了绝对统治力。

**投资洞察**：性能差距的缩小（前几名分差 < 20分）意味着**差异化竞争**将成为下一阶段重点。单纯拼跑分的边际效益递减，推理成本和上下文窗口成为新的决胜点。

### 04. 多模态专区

*   **Google Gemini 3 Flash 领跑多模态应用**：
    `gemini-3-flash-preview` 的高流量（113.9B）很大程度上代表了多模态需求的爆发。Gemini 系列原生的多模态架构使其在处理图文混合任务时具有成本与速度优势。
*   **Molmo-2 与 Amazon Nova 动态**：
    数据库中出现了 `allenai/molmo-2-8b`（输入 $0.19/输出 $0.18）和 Amazon Nova 系列的定价信息。Amazon `nova-lite-v1` 极低的定价（输入 $0.059/输出 $0.238）预示着云厂商正在通过捆绑销售和低价策略，试图在多模态 API 市场分一杯羹。

---

### 05. 价格趋势与成本分析

*   **阿里通义千问极致性价比**：
    `alibaba/tongyi-deepresearch-30b-a3b` 的 API 定价极具侵略性，输入仅 **$0.089/M**，输出 **$0.449/M**，且 Cache Hit Rate（缓存命中率）高达 **69.9%**。这表明阿里云正在通过极高的缓存利用率大幅降低开发者成本，适合高并发、重复 prompt 较多的业务场景。
*   **Amazon Bedrock 入局**：
    Amazon Nova Lite 以低于 $0.06 的输入价格入场，标志着 AWS 正式加入 API 价格战，试图利用云生态优势锁定企业客户。

### 06. 投资建议

1.  **关注国产替代逻辑**：MiniMax、DeepSeek、Stepfun 在流量榜上的统治力证明了国产模型在应用层面的竞争力已不输海外。建议重点关注相关产业链上下游（算力租赁、应用开发）的投资机会。
2.  **警惕价格战风险**：随着 Amazon、阿里云等巨头进一步压低 API 价格，纯 MaaS（模型即服务）厂商的利润空间将被压缩。未来的赢家属于那些能通过“高缓存命中率”优化成本，或提供独特垂直场景数据的厂商。
3.  **技术风向标**：Claude Opus 4.6 的登顶意味着“Thinking/推理增强”模式已成为高端模型的标配。建议关注具备强推理能力的模型在金融、代码等高价值场景的落地进展。
