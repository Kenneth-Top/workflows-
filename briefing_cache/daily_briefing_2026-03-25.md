# 【每日大模型行业情报简报】

**日期：** 2026-03-25
**编制方：** AI 行业投资分析部

---

### 📊 核心观点摘要
今日数据呈现出明显的**“端侧模型爆发”**与**“推理模型霸权”**双重特征。**小米** 凭借 `mimo-v2-pro` 惊人的 2484 亿 Token 消耗量登顶榜首，标志着国产端侧/移动端模型落地已形成规模化效应。**Anthropic** 凭借 Claude 4.6 系列在跑分与用量上双双压制 OpenAI，`claude-opus-4-6-thinking` 以 1502 分刷新 LMArena 记录。价格端，Amazon 与 Alibaba 继续下探底价，推理成本进入“厘时代”。

---

### 1. 核心厂商动态与用量异动

#### 🥇 小米：端侧模型的隐形冠军
*   **数据表现**：`xiaomi/mimo-v2-pro` 以 **2484.43 亿 Token** 的消耗量位居全榜第一，且大幅领先第二名约 40%。
*   **分析师点评**：这不仅是单一模型的胜利，更是“模型+硬件”生态闭环的验证。Mimo 模型的高消耗极有可能源自小米庞大的手机/汽车终端出货量。相比云端大模型，端侧模型的高频次、低延迟特性正在转化为真实的 Token 规模，建议重点关注端侧部署带来的硬件换机需求。

#### 🚀 Anthropic：全面超越 OpenAI 的新霸主
*   **用量数据**：`claude-sonnet-4.6` (1205.9B) 与 `claude-opus-4.6` (1156.1B) 分别位列第 6、7 名，两者合计消耗量已超过 GPT 系列上榜模型。
*   **跑分表现**：在 LMArena 榜单中，`claude-opus-4-6-thinking` 以 **1502 分** 位居第一，`claude-opus-4-6` 紧随其后。相比之下，OpenAI 的 `gpt-5.4-high` 仅以 1485 分位列第 6。
*   **趋势研判**：Anthropic 在代码与复杂推理任务上的优势正在转化为市场份额。Opus 4.6 系列的高昂定价并未阻挡用量，说明其在 B 端高价值场景（如代码生成、复杂分析）中具有不可替代性。

#### 🇨🇳 国产模型第一梯队：MiniMax、DeepSeek、智谱
*   **MiniMax**：`minimax-m2.5` 消耗量达 **1396.28 亿** (第3名)，`minimax-m2.7` 达 **724.20 亿** (第9名)。MiniMax 双模型入围前十，显示出极强的 C 端应用渗透率或 API 调用粘性。
*   **DeepSeek**：`deepseek-v3.2` 消耗量 **1327.99 亿** (第4名)。作为性价比标杆，DeepSeek 依然维持着极高的市场占有率，是中腰部开发者的首选。
*   **智谱 AI (Zhipu)**：`z-ai/glm-5-turbo` 消耗量 **1253.57 亿** (第5名)。GLM-5 Turbo 的稳健表现证实了其在中文企业级市场的稳固地位。
*   **Moonshot (Kimi)**：`moonshotai/kimi-k2.5` 消耗量 **632.36 亿** (第11名)。排名相对靠后，面临头部厂商的激烈挤压，需关注其后续长文本优势的转化情况。

#### 🌐 Google：Gemini 3 系列快速起量
*   **数据表现**：`google/gemini-3-flash-preview` 消耗量 **1085.75 亿** (第8名)。
*   **分析**：Gemini 3 Flash 版本的高用量表明 Google 正在通过激进的定价和性能优化抢占开发者市场，其排名已超越自家旧版 `gemini-2.5-flash`。

### 2. 价格战前沿：极致性价比

今日价格数据显示，模型推理成本进一步下探，主要竞争集中在轻量级模型：

*   **Amazon Bedrock**：`amazon/nova-micro-v1` 价格低至 **$0.034/M (Input)**，成为榜单中成本最低的模型之一，AWS 正在利用基础设施优势构建价格护城河。
*   **Alibaba (通义)**：`alibaba/tongyi-deepresearch-30b-a3b` 报价 **$0.089/M (Input)**，配合高达 72% 的缓存命中率，实际使用成本极具竞争力，主要面向长上下文研究场景。
*   **免费策略**：`stepfun/step-3.5-flash:free` 消耗量高达 **1764.75 亿** (第2名)，证明了“免费策略”在抢占市场份额上的短期爆发力，但需警惕长期算力成本压力。

### 3. 多模态专区

*   **Vision 模型异动**：
    *   **Xiaomi mimo-v2-pro**：作为消耗量最大的模型，其命名中的 "v2" 及高消耗特征暗示其极大概率集成了视觉/多模态能力，且主要应用于移动端实时感知场景。
    *   **Gemini 3 Flash**：Google 的 Gemini 系列原生生支持多模态。`gemini-3-flash-preview` 突破千亿 Token 消耗，意味着多模态处理已从“尝鲜”变为“标配”。
*   **分析师观察**：
    传统的纯文本模型在榜单中的统治力正在被多模态模型稀释。特别是端侧模型的高消耗，往往伴随着图像理解、实时视频流处理等需求。建议投资者关注具备多模态融合能力的端侧芯片及传感器供应链。

### 5. 投资建议与风险提示

*   **重点关注**：
    1.  **端侧 AI 生态**：小米 Mimo 的爆发验证了 AI 手机/AI 硬件的商业化落地，建议关注相关硬件供应链及端侧推理芯片厂商。
    2.  **Anthropic 产业链**：Claude 系列在高端代码与推理市场的统治地位稳固，相关 API 生态及企业级应用集成商值得关注。
*   **风险提示**：
    1.  **价格战持续**：Amazon 和 Alibaba 的超低价策略可能进一步压缩独立模型厂商的利润空间，需警惕中小模型厂商的生存危机。
    2.  **免费模型依赖**：Stepfun 等免费模型的高消耗不可持续，若后续商业化转化不及预期，可能面临算力供给收缩风险。
