### 02. 核心厂商流量与动量分析

根据近 7 天 Token 消耗数据，市场头部效应显著，非核心厂商模型流量迅速边缘化。

#### **🔴 惊人异动：小米**
*   **模型：** `xiaomi/mimo-v2-pro`
*   **数据：** 消耗量达 **4891.46 B**（单位：10亿 Token）。
*   **分析：** 该数值是第二名 `stepfun/step-3.5-flash` 的近 **3倍**。这极大概率意味着小米在其硬件生态（手机、汽车、IoT）中完成了新模型的规模化部署。`mimo` 系列的爆发验证了“端侧模型 + 硬件入口”是目前最高效的流量变现路径。

#### **🟠 国内第一梯队：阶跃星辰 & Minimax & DeepSeek**
*   **Stepfun (阶跃星辰)：** `step-3.5-flash:free` 以 **1663.04 B** 位居第二。标注 `:free` 表明其通过免费策略成功攫取了海量开发者流量，动量极强。
*   **Minimax：** `minimax-m2.7` (1517.03 B) 与 `minimax-m2.5` (932.91 B) 双模型入围前 15，且排名靠前，显示出其模型在 C 端应用或特定 B 端场景中具有极高的粘性。
*   **DeepSeek：** `deepseek-v3.2` 消耗量为 **1409.63 B**，稳居第四。在缺乏免费标签的情况下，依然保持高用量，证明其技术口碑与性价比优势稳固。

#### **🟢 海外巨头：Anthropic & Google**
*   **Anthropic：** `claude-sonnet-4.6` (1207.50 B) 与 `claude-opus-4.6` (1157.95 B) 消耗量接近，显示出用户对其最新一代模型的高接受度。
*   **Google：** `gemini-3-flash-preview` 消耗量 **1149.35 B**，紧追 Claude。Flash 系列作为轻量级主力，在 API 市场份额争夺中表现活跃。
*   **OpenAI：** 值得注意的是，在本次统计的 Top 15 消耗榜单中，未见 GPT 系列模型身影（可能被归类于其他未列出名称或流量分流），这在一定程度上反映了竞争格局的微妙变化。

### 04. 多模态专区

**核心观察：多模态流量高度集中于硬件生态**

本次数据中，多模态模型呈现出明显的“赢家通吃”效应：

1.  **小米 Mimo 系列霸榜：**
    *   `xiaomi/mimo-v2-pro` (Top 1, 4891.46 B)
    *   `xiaomi/mimo-v2-omni` (Top 11, 657.90 B)
    *   **分析：** `mimo-v2-pro` 的命名暗示了视觉与多模态能力。其恐怖的流量数据表明，多模态模型的最大应用场景已从单纯的 API 调用，转向了**手机影像处理、自动驾驶辅助、智能家居视觉**等硬件集成场景。这是纯软件厂商难以比拟的护城河。

2.  **Gemini Flash 的多模态渗透：**
    *   `gemini-3-flash-preview` 与 `gemini-2.5-flash-lite` 均进入消耗榜前列。Google 凭借原生的多模态架构，在处理图文混合请求的成本效率上依然具有优势。

**结论：** 投资者应重点关注具备硬件落地能力的多模态模型厂商。纯 API 形式的多模态模型（如早期的 LLaVA 衍生版）在流量上已无法与硬件集成模型抗衡。

### 05. 价格与成本策略

API 定价出现极端分化，**“极致性价比”与“高端溢价”**并存。

*   **价格战先锋：**
    *   **Alibaba `tongyi-deepresearch-30b-a3b`**：Input 价格低至 **$0.089/M**，Output **$0.45/M**。
    *   **关键指标：** Cache Hit Rate 高达 **63.3%**。这表明该模型通过极高的缓存复用率降低了实际服务成本，是典型的“高并发、低延迟”场景优化策略，旨在抢占中小开发者市场。
    *   **Amazon `nova-micro-v1`**：Input **$0.034/M**，价格几乎免费，AWS 正在通过价格手段防御云市场。

*   **高端溢价：**
    *   **AI21 `jamba-large-1.7`**：Input **$2.0/M**，Output **$8.0/M**。
    *   **Amazon `nova-premier-v1`**：Input **$2.5/M**，Output **$12.5/M**。
    *   这类高价模型通常针对企业级复杂任务或私有化部署场景，与消费级市场形成鲜明对比。

### 06. 投资建议

1.  **关注端侧算力机会：** 小米 `mimo` 系列的爆发验证了 AI 手机/AI 硬件的逻辑。建议重点关注边缘侧 NPU 芯片及端侧模型部署解决方案提供商。
2.  **警惕 API 同质化风险：** 阿里巴巴等厂商将价格压至 $0.1/M 以下，意味着纯文本 API 已成为“水电煤”基础设施，利润空间被极度压缩。投资重心应向具备推理能力的头部模型或具备垂直场景数据的厂商转移。
3.  **Anthropic 估值逻辑强化：** 在跑分榜登顶且流量稳居前列，Anthropic 在技术领先性与商业化落地之间取得了最佳平衡，是目前大模型赛道中最稳健的标的之一。
