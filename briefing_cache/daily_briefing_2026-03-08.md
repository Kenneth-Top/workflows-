【每日大模型行业情报简报】

**日期：** 2026年3月8日
**分析师：** AI 行业投资顾问
**摘要：** 本日数据显现出激烈的市场份额争夺战。Minimax 凭借 `minimax-m2.5` 异军突起，Token 消耗量断层领先；Google Gemini 3 系列在保持高跑分的同时实现了惊人的吞吐量；Anthropic 以 `Claude Opus 4.6` 重夺 LMArena 榜首。国产模型 DeepSeek 与 Kimi 稳居第一梯队，而 OpenAI 在消耗量榜单上的“隐身”与跑分排名的滞后值得警惕。

---

### 一、 核心厂商动态与市场格局

#### 1. Minimax：流量黑洞，M2.5 霸榜
**数据表现：** `minimax/minimax-m2.5` 以 **1999.12 Billion** 的 Token 消耗量位居全球第一，且大幅领先第二名 Google（约 1204.9B），领先幅度达 65%。
**分析师点评：** Minimax 已成为当前 API 市场流量最大的模型提供商。这种断层式的领先通常意味着其拿下了某个超级应用（Super App）或大型企业级 B 端集成订单。考虑到其并未在 LMArena 榜单前十中出现，这表明 Minimax 目前的竞争优势可能更多在于**极具竞争力的定价策略**或**特定的长文本/Agent 场景优化**，而非单纯的基准测试刷榜。

#### 2. Google：Gemini 3 系列全面爆发
**数据表现：**
*   **消耗量：** `gemini-3-flash-preview` 消耗量 1204.9B（排名第 2），`gemini-2.5-flash` 消耗量 580.5B（排名第 10）。Google 旗下模型合计吞吐量极高。
*   **跑分：** `gemini-3.1-pro-preview` 以 **1500分** 位列 LMArena 第三，`gemini-3-flash` 达到 **1473分**。
**分析师点评：** Google 正在执行极其成功的“高质低价”渗透策略。Gemini 3 Flash 模型在跑分上甚至超过了早期的 GPT-5 系列和 Grok 4.1，且承载了巨大的流量，显示出其基础设施的稳健性。Google 已从“追赶者”转变为“全栈领导者”。

#### 3. Anthropic：技术皇冠的持有者
**数据表现：** `claude-opus-4-6` 以 **1504分** 登顶 LMArena 榜首，力压 Google 和 OpenAI。在消耗量上，`claude-opus-4.6` 和 `claude-sonnet-4.6` 分别位列第 5 和第 6，合计消耗超 1600B。
**分析师点评：** Anthropic 证明了其在模型智力上限的统治力。Opus 4.6 的登顶说明其在复杂推理任务上仍是开发者的首选。Sonnet 系列的高消耗则验证了其“主力生产模型”的市场定位。

#### 4. 中国厂商：DeepSeek 与 Kimi 双雄并立
**数据表现：**
*   **DeepSeek：** `deepseek-v3.2` 消耗量 916.1B（排名第 3），稳居全球第一梯队。
*   **Moonshot (Kimi)：** `moonshotai/kimi-k2.5` 消耗量 908.5B（排名第 4），紧随 DeepSeek 之后。
*   **智谱 AI (Zhipu)：** `z-ai/glm-5` 消耗量 363.1B（排名第 14）。
**分析师点评：** 国产模型在全球 API 市场已占据重要席位。DeepSeek 和 Kimi 的消耗量均超过了 Anthropic 的 Opus 和 Sonnet 系列，这主要得益于其在国内市场的广泛应用及极具侵略性的定价。两者在处理长上下文和中文场景上的优势构成了其护城河。

#### 5. OpenAI：沉默的巨人？
**数据表现：** 在 Token 消耗前 15 中，OpenAI 仅有一款 `gpt-oss-120b`（排名第 12，475.2B）和 `gpt-5-nano`（排名第 15，296.4B）入围。LMArena 跑分方面，`gpt-5.4-high` 以 1479 分位列第六。
**分析师点评：** OpenAI 在本日数据中显得异常“低调”。消耗量排名的下滑可能源于两个因素：一是其 API 价格相对竞品仍处高位，导致部分价格敏感型流量流向 Google 或 Minimax；二是其用户更多集中在 ChatGPT C 端应用而非 API 调用。GPT-5.4 跑分未能进入前三，技术代差优势正在缩小。

### 二、 多模态与视觉专区

**重点观测：**
虽然 LMArena 榜单主要反映文本能力，但在 API 定价数据中发现了多模态成本下降的信号：
*   **AllenAI Molmo-2-8b：** 输入价格低至 **$0.199/M**，输出 **$0.197/M**，且 Cache Hit Rate 高达 **98%**。
*   **Amazon Nova-Lite-v1：** 输入价格 **$0.059/M**，输出 **$0.238/M**。

**分析：**
多模态模型正在经历“商品化”过程。Molmo-2 和 Amazon Nova 的极低定价表明，基础的视觉理解能力已不再是高溢价功能。对于投资者而言，这意味着基于 Vision 的应用层创业成本大幅降低，建议关注那些利用低成本多模态能力构建垂直行业工作流的初创公司，而非单纯的基础模型提供商。

### 三、 价格趋势与成本分析

**市场分化明显：**
1.  **高端模型维持溢价：** 如 `ai21/jamba-large-1.7` 输入 $2.0/M，输出 $8.0/M，针对特定企业级需求。
2.  **中端市场血战：** Aion Labs、DeepSeek、Google Flash 系列将价格压制在极低水平。例如 `aion-labs/aion-2.0` 输入仅约 $0.73/M。
3.  **免费/开源模型流量激增：** `stepfun/step-3.5-flash:free` 和 `arcee-ai/trinity-large-preview:free` 进入了消耗量前 15。这表明市场上存在大量对成本极其敏感的开发者需求，他们愿意牺牲部分极致性能以换取零成本。

### 四、 投资建议与风险提示

**建议关注：**
1.  **Minimax 产业链：** 其流量的爆发式增长验证了模型落地能力，建议关注其后续融资动态及生态合作伙伴。
2.  **Google Cloud 生态：** Gemini 3 系列的高性价比使其成为企业迁移上云的首选，利好 Google Cloud 业务营收。
3.  **AI 应用层：** 随着模型价格逼近零边际成本，应用层利润空间打开，尤其是基于 DeepSeek/Kimi 等低成本模型构建的 Agent 应用。

**风险提示：**
*   **OpenAI 技术护城河变浅：** 跑分被 Anthropic 和 Google 反超，需警惕其市场份额被进一步蚕食的风险。
*   **开源模型冲击：** 免费/极低价模型占据大量流量，商业化变现仍是全行业难题，需警惕纯模型提供商的估值回调。
