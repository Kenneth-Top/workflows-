### 🖼️ 多模态专区

**核心动态：**
虽然 LMArena 榜单主要聚焦文本，但 Token 消耗榜揭示了多模态的暗流涌动。

1.  **Xiaomi Mimo-V2-Flash**：以 **505.1B** 的消耗量排名第 12。作为手机端侧或轻量级多模态模型，其消耗量甚至超过了 NVIDIA Nemotron，显示出端侧多模态模型的巨大需求。
2.  **Google Gemini 系列**：Gemini 天生具备多模态能力，`Gemini-3-flash-preview` 的高消耗量（1181.1B）意味着大量的多模态任务（Vision/Audio）正在通过 API 承载。

**分析结论：**
多模态模型正在经历从“炫技”到“实用”的转变。Xiaomi 和 Google 的数据证明，多模态能力正在成为基础模型的标配，且成本正在快速下降。

### 💰 价格与成本趋势

**极端低价案例：**
*   **Alibaba (Tongyi-DeepResearch)**：Input 价格低至 **$0.082/M**，Output 价格 **$0.45/M**。这是目前采样数据中的最低价，且 Cache Hit Rate（缓存命中率）高达 52.5%，进一步降低了实际使用成本。
*   **免费模型策略**：Stepfun 和 Nvidia Nemotron 等提供免费模型，直接拉低了行业的准入门槛。

**市场研判：**
API 市场已进入“血海”竞争阶段。对于非 SOTA（State-of-the-Art）模型，价格已成为唯一的竞争维度。建议关注具备低成本推理能力（如 DeepSeek V3.2）或具备特定场景高粘性（如 Minimax）的厂商。
