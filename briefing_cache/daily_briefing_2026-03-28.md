### 01. 核心观点摘要

今日大模型行业呈现**“端侧模型爆发”**与**“推理模型登顶”**的双重特征。
1.  **用量侧：** 小米 `mimo-v2-pro` 异军突起，以近 4 万亿 Token 的周消耗量断层领先，标志着端侧/私有化部署需求进入爆发期。

### 02. 市场格局与流量异动

基于近 7 天 Token 消耗数据，市场格局发生显著位移，**国产模型与端侧模型占据半壁江山**。

*   **小米 现象级爆发：**
    模型 `xiaomi/mimo-v2-pro` 消耗量达到 **3883.05 B Tokens**，是第二名 Stepfun 的两倍以上。这极大概率与小米近期在手机/汽车端侧部署的 AI 功能（如智能座舱、系统级助手）大规模放量有关。`mimo-v2-omni` 同样进入前 15，验证了小米在多模态端侧的闭环能力。

*   **国产第一梯队稳固：**
    *   **Stepfun (阶跃星辰):** `step-3.5-flash` 以 1739.60 B Tokens 位居第二，显示出极强的 C 端渗透率或 API 调用量。
    *   **DeepSeek:** `deepseek-v3.2` 消耗 1378.29 B，稳居第三，其开源生态与高性价比策略持续生效。
    *   **Minimax:** `minimax-m2.7` 与 `m2.5` 双模型入围前 10，合计消耗超 2400 B Tokens，显示出极高的用户粘性。

*   **海外巨头表现：**
    *   **Anthropic:** `Claude-opus-4.6` 与 `sonnet-4.6` 消耗量均超 1100 B，且 Opus 版本消耗略高于 Sonnet，说明用户愿意为顶级推理能力付费。
    *   **Google:** Gemini 3 系列开始放量，`gemini-3-flash-preview` 消耗 1087.44 B，排名第九。

LMArena 最新榜单（2026-03-28）显示，**推理模型** 已完全主导榜单头部。

### 04. 多模态专区

*   **Vision 级模型动态：**
    本期多模态焦点集中在**小米**与**Google**。
    *   **Xiaomi Mimo V2 系列：** `mimo-v2-pro` 和 `mimo-v2-omni` 的超高消耗量（合计超 4400 B Tokens）是本期最大亮点。这不仅是模型层面的胜利，更是**“硬件+多模态”**落地模式的胜利。数据表明，多模态模型正在从“尝鲜”转向“生产力工具”。
    *   **Google Gemini 3：** `gemini-3-flash-preview` 原生多模态特性使其在处理复杂视觉任务时具备成本优势，消耗量激增 1087 B 证明了市场对高效多模态模型的需求。

*   **投资分析：**
    多模态的竞争主战场正在从“理解图片”向“全端侧交互”转移。小米的案例证明，拥有硬件入口的厂商在多模态时代拥有天然的流量护城河。

### 05. 价格趋势与成本分析

API 市场呈现极端的“两极分化”：顶级模型维持高价，中小模型价格击穿底线。

*   **极致性价比：**
    **阿里巴巴** 的 `tongyi-deepresearch-30b-a3b` 报出惊人低价：输入 **$0.088/M**，输出 **$0.449/M**。更值得注意的是其 **92.59%** 的缓存命中率，这意味着在实际调用中，其成本几乎可以忽略不计。这是针对长文本/搜索场景的“核弹级”定价，将对中小模型厂商形成挤出效应。

*   **云厂商定价策略：**
    **Amazon** 的 `nova-micro-v1` 输入价格低至 **$0.034/M**，定位极其明确——极低成本的简单任务处理。

*   **高价区：**
    **AI21** 的 `jamba-large-1.7` 输入价格仍维持在 **$2.0/M**，**Anthropic** 的顶级模型虽未在采样表中显示具体价格，但依据其品牌溢价，预计仍维持高位。

### 06. 投资建议

1.  **关注端侧算力与模型厂商：** 小米 `mimo` 系列的爆发验证了“端侧大模型”的商业可行性。建议重点关注具备硬件落地能力的模型厂商（如小米、华为）及相关的端侧 NPU 产业链。
2.  **警惕同质化价格战风险：** 阿里巴巴将 30B 级模型价格压至 $0.088/M，对于缺乏差异化能力的中小创业公司是重大利空。投资应回避纯依赖 API 调用的中间层创业项目，转向具备独家数据或垂直场景解决方案的公司。
3.  **锁定“推理”红利：** Anthropic 的 `Thinking` 模型登顶且用量高企，证明市场愿意为“深度思考”能力支付溢价。具备强逻辑/代码能力的模型厂商仍是核心资产。
